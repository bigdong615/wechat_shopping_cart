# 海鲜商城微信小程序

一个功能完整的海鲜商城微信小程序，包含商品浏览、购物车、下单、外送配送、支付等功能。

## 功能特性

### 🏠 首页 - 商品浏览
- 顶部轮播Banner广告位
- 搜索框支持关键词搜索、标签匹配（支持键盘确认搜索）
- 左侧分类导航（活鲜产品、冷冻产品、冷冻称重类、固定包装产品等6大品类）
- 右侧商品列表，支持分类筛选和滚动联动
- 商品标签展示（野生、进口、刺身级等）
- 商品原价/折扣价对比
- 商品加减数量控制（含库存上限检查）
- 规格选择弹窗（多规格商品）
- 库存不足/售罄提示
- 底部购物车悬浮栏（含地址选择）

### 📋 商品详情
- 商品大图展示
- 价格/原价/折扣标签
- 商品描述和标签
- 销售数据（已购人数/关注人数）
- 库存状态显示
- 规格选择
- 服务保障信息（冷链配送/品质保证/坏单包赔/极速发货）
- 底部加入购物车操作栏

### 🛒 购物车
- 商品列表展示
- 单品数量增减
- 全选/单选功能
- 删除商品
- 价格实时计算

### 📦 下单与外送
- 收货地址管理（新增、编辑、删除、设为默认）
- 地区选择器（省/市/区三级联动）
- 配送方式选择（外送配送 / 到店自提）
- 订单备注
- 商品清单确认
- 价格汇总（商品金额 + 配送费）

### 💳 支付方式
- **微信支付** - 微信原生支付
- **支付宝** - 支付宝支付
- 模拟支付流程

### 📋 订单管理
- 订单列表（全部/待发货/待收货/已完成）
- 从个人中心可直接跳转到对应订单状态
- 订单详情页（含物流状态）
- 模拟发货
- 确认收货

### 👤 个人中心
- 用户信息展示
- 订单状态数量角标显示
- 快捷订单入口（直达对应状态）
- 收货地址管理
- 购物车入口（含数量角标）
- 联系客服（拨打电话/在线留言）
- 关于我们
- 清除缓存

## 商品数据

项目包含 **30+** 种商品，覆盖全部分类：
- 活鲜产品：飞蟹、大闸蟹、基围虾、龙虾、扇贝、鲍鱼、生蚝、鲈鱼、多宝鱼、石斑鱼、海参、海胆
- 冷冻产品：北极甜虾、阿根廷红虾、三文鱼、鳕鱼排、扇贝肉、鲍鱼仔
- 冷冻称重类：基围虾、明虾、梭子蟹、花蟹
- 固定包装产品：海鲜礼盒A/B套餐、家庭聚餐装、火锅海鲜拼盘
- 固定包装类：鱿鱼干、虾仁、银鲳鱼、带鱼段
- 肉类/蛋：黑猪五花肉、猪排骨、澳洲肥牛卷、和牛牛排、土鸡蛋、有机鸡蛋

## 使用说明

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具，选择"导入项目"
3. 选择 `seafood-mall` 目录
4. 填入自己的小程序 AppID（或使用测试号）
5. 点击"确定"即可预览

## 数据说明

本项目使用本地存储（wx.setStorageSync）模拟后端数据：
- 商品数据：`utils/data.js` 中定义（使用网络图片URL）
- 购物车：存储在 `cart` 键中
- 地址：存储在 `addressList` 键中
- 订单：存储在 `orders` 键中

## 注意事项

- 商品图片使用阿里CDN网络图片，需要在小程序后台配置 `img.alicdn.com` 为合法域名
- 支付功能为模拟支付，实际接入需要后端支持和微信支付商户号
- 支付宝支付在微信小程序中需通过H5页面跳转实现，当前为演示模式
- 建议将 `project.config.json` 中的 `appid` 替换为自己的小程序 AppID
