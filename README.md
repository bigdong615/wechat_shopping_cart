# 海鲜商城微信小程序

一个功能完整的海鲜商城微信小程序，包含商品浏览、购物车、下单、外送配送、支付等功能。

## 功能特性

### 🏠 首页 - 商品浏览
- 左侧分类导航（活鲜产品、冷冻产品、冷冻称重类、固定包装产品等）
- 右侧商品列表，支持分类筛选
- 搜索功能
- 商品加减数量控制
- 规格选择弹窗（多规格商品）
- 库存不足/售罄提示
- 底部购物车悬浮栏

### 🛒 购物车
- 商品列表展示
- 单品数量增减
- 全选/单选功能
- 删除商品
- 价格实时计算

### 📦 下单与外送
- 收货地址管理（新增、编辑、删除、设为默认）
- 配送方式选择（外送配送 / 到店自提）
- 订单备注
- 商品清单确认
- 价格汇总（商品金额 + 配送费）

### 💳 支付方式
- **微信支付** - 微信原生支付
- **支付宝** - 支付宝支付
- 模拟支付流程

### 📋 订单管理
- 订单列表（全部/待发货/待收货/已完成）
- 订单详情页
- 模拟发货
- 确认收货

### 👤 个人中心
- 用户信息展示
- 快捷订单入口
- 收货地址管理
- 购物车入口
- 清除缓存

## 使用说明

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具，选择"导入项目"
3. 选择 `seafood-mall` 目录
4. 填入自己的小程序 AppID（或使用测试号）
5. 点击"确定"即可预览

## 数据说明

本项目使用本地存储（wx.setStorageSync）模拟后端数据：
- 商品数据：`utils/data.js` 中定义
- 购物车：存储在 `cart` 键中
- 地址：存储在 `addressList` 键中
- 订单：存储在 `orders` 键中

## 注意事项

- 图片目前使用占位符，实际使用时请替换为真实商品图片
- 支付功能为模拟支付，实际接入需要后端支持和微信支付商户号
- 支付宝支付在微信小程序中需通过H5页面跳转实现，当前为演示模式
- 建议将 `project.config.json` 中的 `appid` 替换为自己的小程序 AppID
